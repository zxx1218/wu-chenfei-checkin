# 奶茶记录系统 - 多人统计与对比功能

## 功能概述

奶茶记录系统现已支持多用户（小菲和zxx）的分别统计和趣味对比分析。

## 主要功能

### 1. 个人统计视图
- **独立数据展示**：每个人都有自己的统计数据页面
- **品牌偏好分析**：显示每个人的最爱奶茶品牌
- **健康分数计算**：基于喝奶茶频率和自律天数的个人健康评分
- **近7天趋势**：个人最近一周的喝奶茶情况

### 2. 整体统计
- **总计数据**：两人合计的奶茶杯数和乖乖天数
- **本周预算**：共同的周预算跟踪
- **品牌趋势图**：两人整体的品牌分布柱状图
- **健康分析**：综合两人的健康评分

### 3. 趣味PK对比 ⚔️
新增的对比功能包括：

#### PK排行榜
- **总杯数对比**：谁喝的奶茶更多 🧋
- **健康分数对比**：谁更健康 💪
- **自律天数对比**：谁更克制 🌟
- **日均杯数对比**：谁更节制 📊

#### 个人统计卡片
- 每个人的详细数据统计
- 使用不同颜色区分（小菲-粉色，zxx-蓝色）
- 显示最爱品牌及次数

#### 趣味数据分析
- 🏆 喝奶茶最多的一天谁领先
- 💚 谁更自律（不喝奶茶的天数）
- 🎯 两人的品牌喜好差异
- 📈 合计数据和平均值

### 4. Tab切换界面
用户可以通过三个Tab轻松切换视图：
- **整体统计**：查看两人的综合数据
- **小菲**：查看小菲的个人数据
- **zxx**：查看zxx的个人数据

## 使用方法

1. **记录奶茶时选择饮用者**
   - 在打卡表单中选择"小菲"或"zxx"
   - 填写品牌和奶茶名称
   - 可选上传照片

2. **查看统计数据**
   - 点击Tab切换到不同视图
   - 整体统计：查看综合数据和PK对比
   - 个人视图：查看专属统计和健康分析

3. **趣味对比**
   - 在"整体统计"Tab中滚动到"两人PK"部分
   - 查看各项指标的对比结果
   - 了解谁是"奶茶王者" 👑

## 数据统计逻辑

### 健康分数计算
```
健康分数 = (自律天数 / 总天数) × 100 - (日均杯数 × 10)
范围：0-100分
```

### 对比指标
- **总杯数**：越多越"厉害"（反向指标）
- **健康分数**：越高越好
- **自律天数**：越多越好
- **日均杯数**：越少越好
